# 🌱 SuaVida

SuaVida é uma plataforma de desenvolvimento pessoal que permite registrar diariamente seus estudos, hábitos, humor e energia — ajudando você a visualizar sua evolução ao longo do tempo.

## 🚀 Acesse o projeto

👉 [suavida-eight.vercel.app](https://suavida-eight.vercel.app)

## ✨ Funcionalidades

- Cadastro e login com autenticação JWT
- Registro diário com estudos, humor, energia e notas
- Gerenciamento de hábitos personalizados
- Histórico de registros
- Interface responsiva

## 🛠️ Tecnologias

**Frontend**
- React + Vite
- Axios
- React Router DOM

**Backend**
- Node.js
- Express
- JWT (autenticação)
- bcryptjs (criptografia de senhas)

**Banco de dados**
- PostgreSQL via Neon

**Deploy**
- Frontend: Vercel
- Backend: local (em breve Render)

## 📁 Estrutura do projeto

suavida/
├── api/          → Serverless Functions
├── frontend/     → React (frontend)
├── backend/      → Servidor local para desenvolvimento
└── README.md

## 🔧 Como rodar localmente

```bash
npm run dev
```

Isso sobe o backend e o frontend ao mesmo tempo!
